    7. Import Keywords (`from`, `import`, `as`)
    8. Exception Handling Keywords (`try`, `except`, `raise`, `finally`, `assert`)
  - Using the `keyword` module to list and verify keywords
    - `keyword.kwlist`
    - `keyword.iskeyword()`
